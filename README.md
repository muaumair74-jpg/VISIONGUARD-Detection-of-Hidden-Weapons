# VISIONGUARD-Detection-of-Hidden-Weapons
The "Vision Guard" project is made to solve a problem. This problem is finding weapons in places that need to be very secure. Most security systems do not catch threats that are not easy to see. To fix this the "Vision Guard" project only looks for weapons. It uses sets of data that include thermal imaging. These images show patterns that help the system find objects hidden under clothes. This gives us a security advantage that we did not have before.

The "Vision Guard" system uses the YOLOv11 model. This model is trained to work with images. It can do instance segmentation, which means it can create an image of hidden weapons. This model is better than detection tools. It does not just tell us where an object is it clearly defines the shape of the threat.

The "Vision Guard" project also uses logic to handle security responses. The system has been tested a lot to make sure the thermal camera feeds and the YOLOv11 model work together. The "Vision Guard" is a tool that helps security teams spot and stop hidden threats.

It combines processing with thermal technology to set a better standard for security. This provides a way to improve how we protect buildings and public areas. The "Vision Guard" system makes sure we can catch threats by constantly monitoring. This helps make our public and private places safer for everyone. The "Vision Guard" project is important, for our safety.
